Learning Goals

Students will learn fundamental concepts in machine learning and by the end of the course will be able to:
  • Derive analytical solutions for mathematical fundamentals of ML (partial derivatives, matrix multiplication)
  • Identify the relationship between regularization and training size on training and test accuracy
  • Derive and implement learning algorithms
  • Bias-variance tradeoff
  • Implement a backpropagation algorithm by hand
  • Bias-variance tradeoff

    • In regards to application of machine learning to a data science projects, students will leave the course able to:
  • Identify important machine learning problems
  • Technically-proficient writing
  • Verbally explain problems

    • Grading

  • Homeworks (4) — 40%
  • "Celebrations of learning" (2) — 20%
  • Project — 30%
  • Class engagement — 10%


    • The project grade (30%) is further broken down to:
  • Idea proposal — 2.5%
  • Paper selection — 2.5%
  • Literature review — 5%
  • Project plan — 5%
  • Final presentation (poster session) — 5%
  • Final report — 10%

    • Class engagement grades are a composite of non-graded assignments including but not limited to: (a) in-class worksheets, (b) posting on Piazza, (c) writing up scribe notes.

    Homework deadlines are strict. Homework not turned in ontime will NOT be accepted.

    There will be two ``celebrations of learning." These will be in-class, closed-book exams.
